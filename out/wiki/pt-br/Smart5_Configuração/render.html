Este guia descreve o processo de configuração do dispositivo Smart5 IP54, incluindo:

  • Criação e envio de configurações
  • Atualização de firmware
  • Alternativas via SMS

 

¶ Modo Avançado

Para o Smart5 IP54, é necessário utilizar o Modo Avançado.

  • Acesse pela barra superior → Modo avançado
  • Permite controle detalhado de parâmetros e dados

? Observação: a versão padrão do Device Center não possui suporte completo ao Smart5.

 

¶ Tipos de Arquivos

 

Esses arquivos são utilizados para configuração e atualização do dispositivo.

 

¶ Configuração via SMS

O dispositivo também pode ser configurado remotamente via comandos SMS.

  • Configuração inicial remota
  • Ajustes rápidos em campo
  • Troubleshooting sem acesso físico

? Requer consulta à documentação específica de comandos SMS.

 

¶ Atualização de Firmware

Manter o firmware atualizado é essencial para garantir estabilidade e novas funcionalidades.

 

¶ Atualização Automática

Quando o Device Center detecta firmware desatualizado:

  1. Conectar o dispositivo
  2. Sistema sugere atualização
  3. Clicar em Atualizar firmware

 

Observações

  • O dispositivo não envia dados durante a atualização
  • Não requer internet
  • Necessário Device Center atualizado

 

¶ Atualização Manual

 

¶ Método 1 — Arquivo (.efw*)

  1. Conectar ao dispositivo
  2. Selecionar Connect → Send FW
  3. Escolher o arquivo de firmware
  4. Iniciar atualização

 

¶ Método 2 — Pacote (.fwp)

  1. Conectar ao dispositivo
  2. Selecionar Connect → Send FW
  3. Selecionar pacote .fwp
  4. O sistema envia automaticamente os arquivos necessários

 

¶ Atualização OTA (Over-the-Air)

Atualização remota via rede celular.

  • Dispositivo conectado à alimentação
  • Conectividade com rede celular
  • Integração com sistema de gestão de frotas

? O processo depende da plataforma utilizada.

 

¶ Boas Práticas

  • Utilizar sempre a versão mais recente do firmware
  • Garantir compatibilidade entre Device Center e firmware
  • Evitar interromper o processo de atualização
  • Validar configuração após envio

 

¶ Observações Técnicas

  • Durante atualização, o dispositivo não transmite dados
  • Incompatibilidade de versões pode impedir leitura/escrita de configuração
  • O uso do modo avançado é obrigatório para este modelo
