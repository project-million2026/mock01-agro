Segue passo a passo de cada pagina, como fazer as configurações na ferramenta e Virloc.

¶  Login

Ao acessar o link para a plataforma, a seguinte tela será exibida:

Nesta tela, é necessário inserir suas credenciais para fazer login na plataforma.

Caso ainda não possua um cadastro, entre em contato com o suporte para obter acesso.

Link para a plataforma

 

¶ Inicial

Após realizar o login, você será redirecionado para a página inicial da plataforma:

 

Nesta página, diversas informações essenciais são apresentadas:

Total de Arquivos: Quantidade total de arquivos embarcados na plataforma.

Log de Atividades da Semana: Registro das atividades de login recentes.

Total de Dispositivos: Número total de dispositivos cadastrados.

Envio do Mês: Contagem dos envios realizados no mês atual.

 

¶ Menu Lateral

No lado esquerdo da tela, há um menu de navegação dividido em duas seções para facilitar o acesso às funcionalidades:

 

¶ Administrador (Acesso Master Empresa)

Lista de Usuários: Gerenciamento dos usuários cadastrados na plataforma.

Log de Atividades: Monitoramento do histórico de acessos e ações.

 

¶ Menu Principal (Acesso Comum)

Alterar Credenciais: Opção para modificar senha e informações de login.

Dispositivos: Adicionar, visualizar e gerenciar dispositivos.

Grupo de Dispositivos: Adiciona dispositivos em grupos específicos.

Arquivos: Embarca arquivos de Scripts/Bibliotecas, Firmwares e Áudios.

Envio de Arquivos: Embarca arquivos adicionados na plataforma para os dispositivos.

Envio de Comando Livre: Envio de comandos personalizados para os dispositivos.

 

¶ ? Lista de Usuário

A página de Lista de Usuários é acessível apenas para contas administradoras da empresa.

Nesta seção, é possível adicionar usuários comuns ao grupo da empresa. Para isso, basta clicar no botão Adicionar.

¶
➕ Cadastro de Novo Usuário

Um formulário será exibido para preenchimento.

Os campos obrigatórios para cadastro são:

Nome do Usuário

E-mail

Senha

Repetir a Senha

Após preencher os dados, clique no botão Adicionar para concluir o cadastro do novo usuário.

 

¶ ? Log de Atividades

A página de Log de Atividades registra todas as ações realizadas pelos usuários da plataforma.

 

Nesta seção, é possível visualizar detalhes como:

Data  da ação;

Nome do Usuário que realizou a ação;

Nome da Emprepsa que realizou a ação;

Tipo de Ação (exemplo: criação de usuário, alteração de credenciais, arquivo embarcado e envio de arquivos);

Item - qual arquivo, dispositivo ou usuário que foi criado 

ID - últimos quatro dígitos do numero Serial

Detalhes Específicos da atividade realizada (exemplo: nome do usuário cadastrado e número serial do dispositivo adicionado);

 

¶ ? Alterar Credenciais

A página de Alterar Credenciais permite a modificação dos dados de acesso de um usuário.

Ela é semelhante à página de cadastro de novos usuários, onde é possível alterar qualquer um dos elementos apresentados.

Após realizar as alterações desejadas, clique no botão Salvar para atualizar as credenciais do usuário.

 

¶ ? Dispositivos

A página de Dispositivos exibe todos os dispositivos cadastrados na plataforma.

 

Nesta seção, há diferentes funcionalidades disponíveis:

Importar Planilha: Permite o cadastro de múltiplos dispositivos simultaneamente.

Configurar Tabela: Define quais informações dos dispositivos serão exibidas.

Adicionar Dispositivo: Adiciona um novo equipamento à plataforma manualmente.

 

¶ ➕ Adicionar Dispositivos

 Após clicar em Adicionar, a tela de cadastro do dispositivo será exibida:

Nesta seção, é necessário preencher as informações do equipamento. 
Os campos obrigatórios são:

Serial Único - cada equipamento tem seu numero serial na etiqueta do equipamento

Identificador (ID do dispositivo)

 

AVISO ⚠️: NO SERIAL UNICO COLOCAR TODOS OS CARACTERS DO NUMERO SERIAL DO EQUIPAMENTO

 

Além disso, podem ser preenchidos outros campos opcionais.

Após inserir os dados, clique em Adicionar para concluir o cadastro do novo dispositivo.

   

Ao adicionar o equipamento na plataforma, é necessário fazer algumas configurações no equipamento:

  1. Apontar o equipamento para o servidor:

    VL8, VL12 e VC07 → VSIPx,163.176.250.223.4096.XXXXX

    VL6 → SIPx,163.176.250.223,XXXXX,00,F

    x → Socket do IP selecionado

    XXXXX  → Porta da empresa
     
  2. Configurar um Keep Alive enviando ao servidor:

    Exemplo:

    VL8, VL12 e VC07 → SEDxx TT30++ +- GFx TT

    VL6 → SEDxx TT15++ +- ZGx TT
     
  3. Após cadastrar o equipamento na plataforma e fazer as configurações necessárias o equipamento precisa estar com o status de “ONLINE”.

 

¶ ?️ Grupo de Dispositivos

A página de Grupo de Dispositivos permite a criação de grupos para facilitar o gerenciamento e envio de comandos em lote.

Basta inserir um Nome para o Grupo e adicionar os dispositivos desejados.

 

¶ ? Arquivos

Nessa parte é possível embarcar arquivos de Scripts/Bibliotecas e Firmwares.

*As abas de Motoristas, Cercas e Áudios ainda não estão disponíveis*

 

¶ ? Envio de Arquivos

A página de Envio de Arquivos permite registrar e acompanhar os envios realizados.

 

Nesta seção, é possível:

Visualizar os envios em processo e finalizados.

Criar um novo envio clicando em Novo Envio.

 

¶ Criando novo envio

Ao clicar em Novo Envio, a seguinte tela será exibida:

Selecionar o Script Embarcado a ser enviado.

Escolher o(s) Dispositivo(s) que receberão o envio.

Confirmar o envio para processá-lo.

 

Esse processo se repete ao fazer o envio de firmware e áudio

 

¶ Envio de Firmware e áudio

A página de Envio de Arquivos na Aba de Firmware ou áudios  permite registrar e acompanhar os envios realizados.

AVISO⚠️: OS ARQUIVOS DE AUDIOS E FIRMWARES PRECISAM ESTAR EM FORMADO SFB

Nesta seção, é possível:

Visualizar os envios em processo e finalizados.

Criar um novo envio clicando em Novo Envio.

 

Selecionar o Firmware Embarcado a ser enviado.

Escolher o(s) Dispositivo(s) que receberão o envio.

Confirmar o envio para processá-lo.

É o mesmo processo para envio de Audio

 

¶ Comandos livres

Na lateral esquerda na opção de envio de comandos livres, é possível enviar configurar o equipamento remotamente

 

Nesta seção, você irá em Novo Envio e após irá selecionar o equipamento que deseja(é importante que o rastreador esteja conectado a plataforma)

Os ID com a bola verde são os equipamentos conectados

Após selecionar, você pode enviar comando remotamente
