¶ Embarcar Áudios Personalizados em Equipamentos MDVR/Dashcam

A funcionalidade de áudio embarcado permite a troca de sons por áudios personalizados, para efetuar a troca de áudios é necessário utilizar um pen drive.

 

Requisitos

Para que o equipamento aceite os áudios importados, os arquivos precisam seguir requisitos muito específicos. Se uma dessas regras não for seguida, a importação falhará.

  • Formato: .wav
  • Tamanho: até 100kb
  • Canal: Single
     

¶ ? Importando os arquivos no equipamento

Faz a importação automática dos arquivos. Para utilizar esse modo, seguir o seguinte passo-a-passo:

  1. No pen-drive, criar o seguinte caminho de diretório: autoupgrade/mp3/host/
  2. Copiar todos os arquivos .wav para dentro da pasta host
  3. Com o pen drive pronto, basta conectá-lo na porta USB do equipamento
  4. Utilizando o aplicativo Veyes App, buscar e selecionar a opção “Import MP3 File”

¶ ? Restaurando os áudios para padrões de fábrica

Para restaurar os áudios para padrões de fábrica, selecionar a opção "Restore Default Mp3 File".

Ao clicar nela, todos os áudios personalizados serão apagados e os sons padrão serão restaurados.
