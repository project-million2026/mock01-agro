¶ Introdução - Gateway CAN

O hardware Gateway CAN e equipamentos VIRLOC com as funcionalidades de Gateway CAN.

Estas funcionalidades permitem que as bibliotecas CAN e comandos XVM (com sintaxe compatível com VL06) possam ser gravados e atualizados através de arquivos binários (arquivos CLIB com extensão .SFB).

Assim a documentação abaixo descreve as funcionalidades e processos para manipulação destes recursos.

 

¶ Hardware

Conexões físicas baseadas no contra-cabo de 8 vias do VL06 ou VL08.

 

 

¶ Arquivos de Biblioteca CAN - CLIB File

O Gateway CAN permite o uso de bilbiotecas CAN normalmente, programadas em XVM como qualquer outro Virloc. Porém também permite que as configurações sejam embarcadas através de um arquivo SFB criptografado.

O arquivo CLIB é um arquivo de texto, contendo um cabeçalho e comandos XVM para serem posteriormente convertidos para SFB e instalados em dispositivos canGW ou VL06.

 

¶ Nome do arquivo

Todos os comandos XVM para a criação de uma biblioteca CAN e demais comando necessários devem estar contidos em um arquivo de texto com extensão .txt.

O nome do arquivo deve obedecer o seguinte formato:

CLIBxxyy.txt

Onde:

  • xxyy = uma sequência de 4 caracteres numéricos livres ao usuário, que podem ser utilizados como forma de versionamento, por exemplo.

¶ Ativar bibliotecas

Para ativar uma bilbioteca, primeiro verifique se ela foi carregada corretamente com a consulta >CLIB<.

Exemplo de retorno:

>CLIB,0001,0002,lines:18;ID=3186;#818F;*08<

Onde:

  • 0001 = ID da primeira bilbioteca (conforme configurado no arquivo)
  • 0002 = ID da primeira bilbioteca
  • …
  • 18 = Quantidade total de linhas das bilbiotecas

 

Para ativar a biblioteca, utilize o comando CLIB_RUN conforme os exemplos:

>CLIB_RUN< // Ativa todas as bilbiotecas embarcadas
>CLIB_RUN,0001< // Ativa a bilbioteca ID 0001
>CLIB_RUN,0001,0002< // Ativa as bilbiotecas ID 0001 e 0002

Exemplo de retorno:

>CLIB_RUN,0001,lines:11;ID=3186;#8190;*4E<

Onde:

  • 0001 = ID da primeira bilbioteca (conforme configurado no arquivo)
  • 11 = Quantidade de linhas executadas

 

Observação importante:

Quando são ativadas, as bibliotecas não apagam automaticamente os filtros que não são utilizados. Portanto garanta que as suas bibliotecas contenham os filtros sobressalentes vazios.
