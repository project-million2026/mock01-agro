¶ Hardware - M1N

A M1N contempla as seguintes especificações.

 

 

¶ Dimensões

154*151*52 mm

 

¶ Peso

1,7 Kg

 

¶ Painel Frontal

 

1 - Interface USB (Porta USB 2.0)

2 - Interface de Comunicação (1 × Slot para cartão SIM)

2 - Interface de Armazenamento (2 × Micro SD + 1 × SSD M.2 SATA)

3 -  Trava do Disco Rígido (Sistema de bloqueio do disco)

4 - Indicador LED 

 

¶ Painel Traseiro 

 

 

POWER → Conector de Alimentação

A/V IN1-4 → Entradas de Vídeo/Áudio AHD

A/V OUT → Saídas de vídeo/áudio

IPC → Entrada de vídeo/áudio IPC

Entrada para Antena GNSS

Entrada para Antena WIFI

MAIN → Antena principal 4G/3G

IPC2/AVOUT/RS485 → Interface externa IPC2 / saída de vídeo HD / RS485

IO/RS232 → Interface externa IO / RS232
