¶ Introdução

 

A JC170 é uma câmera com inteligência artificial embarcada capaz de detectar sinais de distração e fadiga do motorista. Por possuir uma serial TTL, ela pode ser utilizada como periférico aos equipamentos da linha Virloc, como VL06, VL08 e VC07, assim podendo enviar os pacotes e alertas da câmera como dados livres ao servidor, ou tomar alguma ação com base nesses alertas. Este guia tem com objetivo configurar a JC170 para funcionar adequadamente com os Virlocs.                              

                                                    

¶ Procedimento

Para o envio de comandos para a câmera, será necessário um monitor serial, neste guia foi utilizado o Docklight.

1. Enviar comando AT^GT_CM=VERSION para verificar versão de firmware da JC170, o equipamento responderá:

[VERSION]C170_MB_WABL_V2.3_230620180001

Onde:

V2.3 - Versão de firmware da JC170 (deve ser igual ou superior a versão de firmware 2.3)

Obs: Caso o equipamento esteja em uma versão de firmware inferior a 2.3, entre em contato com o suporte da Jimi para realizar a atualização.

 

2. Enviar comando AT^GT_CM=C170SYS,3,1 para configurar a JC170 em modo ASCII:

 

3. Alterar a baudrate para 9600 com o comando AT^GT_CM=C170SYS,2,9600 (necessário apenas para VL06, pois sua baudrate máxima é 19200. Para VL08 e VC07, esse passo não é necessário).

 

Feito isso, a câmera já estará apta para enviar os alertas via Serial para o Virloc.

 

¶ Testando comunicação com a câmera

Ajustar Baudrate do equipamento:

>VS16,BE,9600< (VL06)

>SRS1 9600< (VL08,VC07)

Onde:

9600 = Baudrate a ser configurado 

Com o equipamento conectado via bluetooth, envie o comando SDB10 para ativar o debug da porta serial e depois gere um alerta na câmera. Os dados livres dos alertas chegarão no Virloc, conforme imagem abaixo:

 

¶ Exemplo de script

VL06:
>SUC03 QTT,4,99 GX1,5,99< //Pacote RUV03 com os dados livres da serial
>SED030 T100++ +- ZG0 V3< //Envia para o IP0 os dados livres da câmera sempre que chegar um novo alerta
>VS16,BE,9600< //Baudrate da serial

 

VL08/VC07:
>SXT0010010101< //Habilita o protocolo TXT (Dados em ASCII)
>SUC03 QTT,4,99 QXB00,7,99< //Pacote RUV03 com os dados livres da serial
>SED030 XT10++ +- GF0 V3< //Envia para o IP0 os dados livres da câmera sempre que chegar um novo alerta
>SRS1 9600< //Baudrate da serial
