¶ Envio de Comandos via SMS

Para enviar comandos de configuração e consulta ao VIRLOC6 é necessário cadastrar os números que serão permitidos. Caso não haja nenhum número cadastrado, o VIRLOC irá aceitar SMS de qualquer origem, porém não irá enviar respostas como confirmação dos comandos.

 

¶ ? Cadastrando Números

Podem ser cadastrados até 2 números distintos, e para isso utiliza-se os comandos:

>SIS28"numero1"<

>SIS29"numero2"<

 

? Para consultar os números cadastrados, basta enviar:

>QIS28<

>QIS29<

 

¶ ⚙️ Enviando Comandos e Consultas

Para enviar consultas ou comandos XVM para o VIRLOC, basta enviar a mensagem precedida de um “>”.

Podem ser enviados quaisquer comandos/consultas XVM, com exceção do CLN.

¶ ✉️ Report via SMS

Também é possível programar o VIRLOC para enviar reports via SMS, sendo que os destinos serão os números configurados em IS28 e IS29. Para referenciar esses destinos, utiliza-se MA0 ou MA1 respectivamente.

? Exemplo:

>SIS28"11986994352"< //Configura número do destino MA0
>STD010060< //Configura o TD com tempo de disparo a cada 60s
>SED001 TD01++ +- MA0 GP< //Evento que realiza o envio de uma mensagem GP a cada disparo de TD01
