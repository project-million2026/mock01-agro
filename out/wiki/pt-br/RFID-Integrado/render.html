¶ ? Leitor RFID - VIRLOC 08 / VIRCOM 07 

Os equipamentos VIRLOC 08 (VL08) e VIRCOM 07 (VC07) podem contar, como opcional, com um leitor RFID integrado para cartões Mifare 13,56 MHz.

Esse leitor facilita soluções que exigem identificação de condutores e/ou passageiros, reduzindo complexidade de instalação, cabeamento e integração.

¶ ⚙️ Configuração do leitor RFID

Para a utilização do leitor, o mesmo pode ser habilitado e configurado na programação XVM através do comando:

 

>SMF1,00,00,06,0,0440,005,100,002,4,MF1,IC0<

Onde: 

SMF1 = ativa leitor RFID (para desativar SMF0)

00 = Bloco de leitura do código na memória cartão 

00 = Offset para capturar código na memória cartão 

06 = Quantidade de bytes para capturar (lenght)

0 = número de chave a utilizar (cadastradas em SMK)

0440 = Frequência som a reproduzir ao ler cartão no VIRCOM 07; para VIRLOC 08 reproduzirá o áudio embarcado com o nome READ

005 = Delay entre escaneamento de cartões [0 a 244, sendo 100 = 1s]

100 = Delay entre leitura de cartões [0 a 244, sendo 100 = 1s]

002 = Tempo para completar leitura [0 a 244, sendo 100 = 1s]

4 = Ganho da antena do leitor [0 a 6, sendo 6 o padrão para VL08 e 4 para os demais] - Somente a partir do firmware 155B13

MF1 = Leitura de cartão Mifare, podendo assumir os valores:

  • MF0 - desabilita Mifare
  • MF1 - leitura com inversão de bytes
  • MF3 - leitura sem inversão de bytes
  • MF5 - leitura com inversão de bytes e com checksum
  • MF7 - leitura sem inversão de bytes e com checksum

IC0 = Leitura de cartão iClass, podendo assumir os valores:

  • IC0 - desabilita iClass
  • IC3 - leitura com inversão de bytes

 

¶ ? Chaves de criptografia (SMK)

Para casos onde se utiliza chaves de criptografia nos cartões, pode ser cadastrada com o comando:  

>SMKxAABBCCDDEEFF<

Onde:

x = Numero de chave a configurar [1..4];

AABBCCDDEEFF = Chave em formato hexadecimal.

 

¶ ? Controle de energia do leitor RFID

O leitor RFID Mifare integrado ao VIRLOC pode ser desligado e ligado juntamente com a String de comando VSLP.

>VSLP_MF1<

Onde: 

MF1 = Liga leitor RFID (MF0 desliga leitor RFID)

⚠️ Importante:
Por padrão, quando o VIRLOC/VIRCOM é ligado, o leitor RFID vem desligado.

 

¶ ? Consulta do último cartão lido

Após ativar o leitor e efetuar uma leitura, pode ser consultar o ID do cartão RFID lido utilizando o comando: 

>TPCR_LAST<

 

Cuja a resposta será: 

>PCR_REG02:999999,c00000000A7930C62,99,,2811432034,,,,,,,,2;ID=VL8I,#8200,*53<

Onde: 

A7930C62 = é o número do cartão RFID em Hexadecimal

2811432034 = é o número do cartão RFID em Decimal

 

¶ ? Disparador de leitura de cartão

Sempre que um cartão for lido, o disparador IO03++ será acionado, e pode ser utilizado em um evento, por exemplo:

>SED001 IO03++ +- GF0 TT<

 

¶ ? Exemplo de Código para LOGIN Condutor (VIRLOC 08) 

No exemplo abaixo, é demonstrada uma lógica de login para utilização com VIRLOC 08, utilizando o leitor RFID integrado e também áudios de instrução para o condutor.

Link para download: Áudios MP3 de demonstração

 

**********************
*** LOGIN VIRLOC 8 ***
**********************


>SIT0003003_AD< //DEFINE IN00 COMO ENTRADA ANALÓGICA

//CONFIGURA IGNIÇÃO POR ENTRADA ANALÓGICA IN00 EM PK:
>VSPK,2150,2000,1100,1070,0360,0340,0350,0340,0350,0340,0430,0425,0340,0800,0450,+50,+70,1200,1200,0200,999,A5,001<
//Onde:
// 2500 = Tensão de detecção IGNIÇÃO (quando alimentado 12V);
// 2500 = Tensão de detecção IGNIÇÃO (quando alimentado 24V);
// 0200= Histerese para detecção de IGNIÇÃO;
// A5 = Define o tipo da ignição: ignição física por entrada analógica = A5;
// 001 = Define fonte da ignição: para ignição física por entrada analógica (A5 no parâmetro anterior) seleciona a entrada analógica (001 para AD1 - entrada IN00 / fio marrom).
//
// Assim:
//  Reconhece IGNIÇÃO LIGADA ao ler tensão acima de 12,0 Volts em IN00;
//  Reconhece IGNIÇÃO DESLIGADA ao ler tensão abaixo de 11,8 Volts (12V - 0,2V = 11,8V).

//CONFIGURAÇÕES PARA UTILIZAR ÁUDIOS:
>VSRM12,0,100000,290000< //MEMÓRIA RESERVADA PARA ARQHIVOS MP3 = 200K * 16 = 3 Mbytes
>TCFG04,912< //ÁUDIOS ARMAZENADOS NA REGIÃO DE MEMÓRIA DEFINIDA POR RM12, VOLUME PADRÃO 9

>VSLP_MF1< //LIGA LEITOR RFID

>SMF0< //DESATIVA LEITOR RFID

>SMF1,00,00,06,0,0440,005,100,002,4,MF1,IC0< //CONFIGURA E ATIVA LEITOR RFID
//SMF1 = Ligado
//00   = Bloco de leitura do código
//00   = Offset para capturar código
//06   = Quantidade de bytes para capturar (len)
//0    = número de chave a utilizar
//0440 = Frequência som a reproduzir ao ler cartão
//005  = delay entre scanner de cartões
//100  = delay entre leitura de cartões
//002  = Tempo para completar leitura
//4    = Ganho da antena do leitor
//MF1  = Habilita leitura de cartão Mifare sem inversão de bytes
//IC0  = Desabilita leitura de cartão iClass


>SED000 IN07++ SG01-- SGN NN {PFL,IDENT,0,9}{SSXP3100001B30,2,1}{SSXP4100001B30,2,1}{VSLP_MF1}{SMF1}<
//QUANDO IGNIÇÃO É LIGADA, COM CONDUTOR NÃO LOGADO, É REPRODUZIDO ÁUDIO PARA O CONDUTOR SE IDENTIFICAR E ACIONADO LED VERDE SUPERIOR PISCANDO
//{PFL,IDENT,0,9} = REPRODUZ ÁUDIO PARA O CONDUTOR SE IDENTIFICAR
//{SSXP3100001B30,2,1} = ACIONA LED VERDE SUPERIOR PISCANDO (30 VEZES, 0,2 SEGUNDO LIGADO E 0,1 SEGUNDO APAGADO)
//{VSLP_MF1}{SMF1} = LIGA LEITOR RFID

>SUT01HTPCR_LAST,28,8,0,0<
//CAPTURA O ID DO CARTÃO RFID EM MODO HEXADECIMAL
//EXEMPLO DADO: >PCR_REG02:999999,c00000000A7930C62,99,,2811432034,,,,,,,,2;ID=VL8I;#8200;*53<


>SED001 IO03++ SG01-- SGN NN {SCT01 VUT01}{SCC01 1}<
//SEMPRE QUE UM CARTÃO É LIDO, COM CONDUTOR NÃO LOGADO, COPIA O VALOR DO ID DO CARTÃO PARA O CT01
//IO03++ = disparador cartão RFID lido

>SED002 CC01-- +- SGN NN {VGCT,0,799,1,QCT01,7,10}<
//COMPARA O CARTÃO LIDO COM A LISTA DE CONDUTORES EMBARCADA

>SED003 RC00++ SG01-- SGN NN {PFL,BOAVIAGE,0,9}{SSH011}{SSXP3100001B04,3,1}{VSLP_MF0}<
//CONDUTOR CADASTRADO, SOBE FLAG DE LOGIN E DESLIGA LEITOR RFID

>SED004 RC03++ SG01-- SGN NN {PFL,NCADASTR,0,9}{CCT01}{SSXP4100001B04,3,1}<
//CONDUTOR NÃO CADASTRADO

>SED005 IN07-- +- SGN NN {SSH010}{CCT01}{VSLP_MF0}<
//LIMPA CT01 COM ID DO RFID, ABAIXA A FLAG DE LOGIN E DESLIGA LEITOR RFID.

>VSRM00,2,0,800< //DEFINE REGIÃO 00 EM MODO 2 (BASE DE DADOS) DO PONTO 0 AO 800 PARA LISTA DE CONDUTORES
//PODEM SER DEFINIDOS 800 CONDUTORES (VSRT000...799)

//CONDUTORES:
>VSRT0,2276394850< //CARTÃO 1
>VSRT1,1086388903< //CARTÃO 2

>SUT02,QTT,33,3,0,5<
//CAPTURA VALOR DA VELOCIDADE NA FAIXA DE 0 A 5KM/H PARA DETECTAR MOVIMENTAÇÃO

>SED006 UT02++ +- SGN NN {VSLP_MF1}<
//LIGA LEITOR RFID QUANDO VELOCIDADE É MENOR QUE 5KM/H

>SED007 UT02-- +- SGN NN {VSLP_MF0}<
//DESLIGA LEITOR RFID QUANDO VELOCIDADE É MEIOR QUE 5KM/H
