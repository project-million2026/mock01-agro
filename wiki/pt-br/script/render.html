¶ Embarcar script

 

¶ ? Ligações físicas necessárias (VL6/VL8):

  • Fio vermelho com listra branca → +12~+24 VDC
  • Fio preto → Negativo (GND)
  • Fio amarelo → Garra amarela do cabo USB
  • Fio azul → Garra azul do cabo USB
  • Garra preta do cabo USB → Negativo (GND)

Obs.: Caso vá configurar o equipamento via Bluetooth, dispense as últimas 3 ligações, apenas alimentando o equipamento na fonte e seguir o guia de Bluetooth para correta conexão. Para mais informações das conexões dos Virlocs, veja a página de Entradas e Saídas de cada modelo.

 

¶ ? Carregamento de script

 1. Abra o XVM Terminal (caso ainda não possua instalado, baixe o Novo XVM Terminal).

 2. Com o terminal aberto, selecione a porta COM correta e o baudrate (padrão 19200 para VL06, 115200 para VL08, VC07 e VL12) conforme destacado abaixo, e clique em Conectar.

Você consegue identificar a porta COM correta do equipamento em Gerenciador de Dispositivos do Windows. Para identificar a porta COM correta por Bluetooth, veja a página guia de Bluetooth no passo 8. 

 

 3. Estabeleça comunicação com o equipamento enviando um comando qualquer, como QSN<>QVR pressionando enter na caixa de texto inferior após ter digitado esse comando, ou os botões pré-configurados. (num primeiro instante, talvez seja necessário enviar comandos mais de uma vez para haver resposta, após a primeira resposta, ele seguirá comunicando normalmente).

Após enviar o comando algumas vezes, o equipamento deve responder como a última linha RX: >RVR… indicando o equipamento e versão de firmware.

Caso o equipamento já estava configurado anteriormente, é recomendado enviar o comando CLN00000000_FULL (VL6) ou CLN0000 (VL8, VC7, VL12) para limpar as configurações contidas nele, carregando o script no equipamento zerado.

 

 4. Com a comunicação estabelecida, clique em Embarcar Script e selecione o seu script (geralmente um arquivo .txt).

 

5. Todos os comandos contidos no arquivo de texto serão embarcados no equipamento, e no final deverá ser exibida a mensagem Programação enciada com sucesso! indicando que a configuração foi realizada. Clique em Confirmar para retornar ao terminal.
