O Gateway CAN é um hardware capaz de traduzir as informações da rede can presente em veículos e máquinas para um protocolo de texto simples. Os dados coletados são disponibilizados através de uma porta serial TTL ou RS232 no formato de reportes que serão apresentados ao longo deste documento. 

Este hardware tem como principal objetivo agregar ao sistema de rastreamento ou vídeo monitoramento básico dados de telemetria, possibilidade análises e relatórios mais ricos. Sendo assim o Gateway CAN como periférico faz a ponte entre dados CAN e plataforma.

 

¶ Formato de Mensagem

A comunicação via serial é realizada através do protocolo de texto simples, onde todas as mensagens contêm apenas caracteres em ASCII seguindo o padrão XVM: 

>RUV03102,4,6120,5665,4296,0,0,16,1640,0,42,130,0,0,0,0,0,0,0,0,1;ID=0002;#0306;*79<

¶ Conteúdo da Mensagem

O conteúdo de cada mensagem é fixo e contém um cabeçalho do tipo RUV que indica o tipo de pacote, podendo ser uma mensagem de apresentação ou pacote de dados da rede CAN. A seguir serão apresentados os pacotes e cada informação que compõe a string.

 

¶ RUV00 – MENSAGEM DE APRESENTAÇÃO / CONFIGURAÇÃO 

? Exemplo:

>RUV00100,26N40002,10.16,V1.0;ID=0002;#0363;*10<

¶ RUV03 – DADOS DE TELEMETRIA 

? Exemplo:

>RUV03102,4,5451,4995,1576,0,0,16,2190,0,98,30,0,0,0,0,0,0,0,0,1;ID=0002;#0297;*73<

Campo: 

102 = Natureza da mensagem (consulte tabela INDENTIFICAÇÃO DE MENSAGENS);

4 = Posição do pedal do acelerador (em %);

5451 = Horímetro do motor (minutos); 

4995 = Odômetro total do veículo (metros); 

1576 = RPM do motor (rotações por minuto);

0 = Temperatura do refrigerante do motor (ºC);

0=Pressão do óleo do motor (kPa);

16= Nível de combustível (em %);

2190=Consumo total acumulado (litros x10);

0= Reservado - P1;

98= Velocidade (km/h);

30 = Torque do motor (em %); 

0 = Reservado - P2;

0 = Freio motor (em %);

0 = Reservado - P3;

0 = Reservado - P4;

0 = Reservado - P5.

0 = Cruise Control [0 = OFF | 1 = ON]

0 = Estado da embreagem [0 = OFF | 64 = ON]

0 = Estado do Freio de estacionamento [0 = OFF | 04 = ON]

1 = Estado do Freio de serviço - Pedal [0 = OFF | 08 = Pressionado]
