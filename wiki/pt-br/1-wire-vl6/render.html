¶ OneWire VL06

Este protocolo permite a integração dos Virlocs com iButtons (para identificação) e sensores de temperatura. Utiliza a entrada IN3 e a saída OUT1.

Importante: como são utilizadas as mesmas conexões da porta serial, não é possível utilizar os dois recursos simultaneamente. Para testes, conecte-se ao VL06 via Bluetooth.

 

¶ Habilitar e Desabilitar

Para habilitar o protocolo OneWire:

>VSIB_ENA<

Para desabilitar o protocolo OneWire: 

>VSIB_DIS<

 

¶ Leitura do IButton ID 

Para ler os dados adquiridos pelo iButton, utilizamos o:

>GIO<

 

¶ Disparador 1-Wire

IB00++ → iButton ID lido
 

¶ Conexões físicas

 

¶ ? iButton

iButton: conectar em IN3/OUT1

 

¶ ?️ Sensor de temperatura

Para utilizar um sensor de temperatura OneWire é necessário ativar a saída de tensão auxiliar disponível no fio da entrada digital 02 (IN2 - FUNCIONAL SOMENTE PARA VL06 FULL), para fornecer tensão para o sensor. 

O comando de ativação é o indicado abaixo:

>SSXP21< 

 

¶ Leitura de Sensor de Temperatura

Os dados do sensor de temperatura podem ser lidos através da consulta a seguir:

Comando:

>GIT<

? Exemplo de Resposta:

>GIT00,+0210 R165 C000 H085 L005 TM000000000 VAL:1, 00,280F6680E3E13CE1;ID=0002;#B680;*07<

Onde:

+0215 = Leitura de temperatura em °C x10 (21,5º C no exemplo)

R165 = Este valor é o resto da leitura. Pode ser usado em conjunto com o valor C para obter leituras mais precisas.

C000 = Este valor é a contagem por grau. Pode ser usado em conjunto com o valor rrr para obter leituras mais precisas. Cálculo: Temperatura = ( TM / 2 ) - 0,25 + [ ( C - R ) / C ]

H085 = Registrador de temperatura máxima (dado lido do sensor)

L005 = Registrador de temperatura mínima (dado lido do sensor)

TM000000000 = Tempo desde a última leitura

VAL:1 = 1 leitura válida | 0 leitura inválida

280F6680E3E13CE1= ID do sensor
