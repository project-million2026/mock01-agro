¶ GATEWAY CAN + Wialon

Este procedimento descreve como integrar o GATEWAYCAN com a câmera JIMI JC450 e habilitar o envio dos dados CAN para a plataforma Wialon.

¶ ⚙️ Configuração de Comunicação Serial

Os dois equipamentos devem usar o mesmo baudrate na porta serial.

 

¶ GATEWAYCAN

 

? Para mais detalhes de configuração do aplicativo, consulte: /app-gatewaycan

 

¶ JC450

 

¶ ? Habilitar comunicação serial (UART TTL)

UART,TTL,ON,0000,ON,19200,,

Onde:

TTL = Uso da porta TTL interna;

ON = Habilita a UART;

19200 = baudrate (deve ser igual ao configurado no GATEWAYCAN).

 

¶ ? Configuração de APN

APN,NOME,APN,,,,,,LOGIN,,SENHA,,,,,IP,IP,

Onde:

NOME = Nome que deseja configurar para essa APN;

APN = Apn que deseja configurar;

LOGIN = Login que deseja configurar;

SENHA = Senha que deseja configurar;

IP = Define a comunicação em IPv4 (Não alterar).

 

¶ ?️ Configuração de Servidor

SERVER,64.120.108.24,22305,NA,NA,NA,NA

Onde:

64.120.108.24 = IP do servidor;

22305 = Porta de Destino.
