¶ Configuração Básica

 

¶ ? Preparação

Para realizar qualquer consulta ou configuração no equipamento pelo celular será necessário baixar e instalar o app Veyes.

 

Após obter o aplicativo, ligue o equipamento com o fio ACC no positivo, e olhe as redes WiFi disponíveis no seu celular, uma rede com o SN do equipamento, ou com a placa do veículo estará disponível, estabeleça uma conexão com esta rede.

 

Após se conectar na rede, abra o aplicativo Veyes. A seguinte tela de login irá aparecer:

Insira seu usuário/senha e aperte em “Login”

*O usuário e senha padrão será: 

admin/admin 

ou 

admin e senha em branco

 

¶ ⚙️ Configurações

 

¶ Informações do Veículo

Preferências → Informações de Registro

Nesta tela é possível colocar as informações do veículo onde o equipamento está instalado. 

O campo número de matrícula altera o nome da rede Wifi gerada pelo equipamento quando o mesmo inicia, é recomendado colocar o mesmo nome da placa do veículo, ou outra informação que identifique o veículo.

 

¶ Definição de Data/Hora

Preferências → Definição de Tempo → Geral

A partir desta tela é possível definir o formato da data/hora e o GMT do equipamento.

 

Preferências → Informações de Registro → Ajuste de Tempo

Por meio desta tela, é possível ajustar manualmente, ou automaticamente o tempo. É recomendado que se utiliza o ajuste automático.

  • O ajuste automático por satélite, irá ajustar o horário de acordo com as informações obtidas pelo GNSS
  • O ajuste por NTP, irá ajustar o horário baseado na rede WAN
  • O ajuste por centro, irá requisitar o horário do servidor conectado

Caso mais de um ajuste automático seja selecionado, o equipamento seguira a ordem de prioridade: Satélite → NTP → Centro para ajustar o horário.

 

Preferências → Definição de Tempo → Horário de Verão

Nesta tela possível definir um período para o horário de verão, assim o equipamento automaticamente aplicara o defasamento de hora no período indicado.

 

¶ Definição de Delay de Shutdown

Preferências → Definição Básica → Definição ao Ligar/Desligar → Ligar/Desligar

Nesta tela é possível definir por quanto tempo o equipamento ficara ligado caso a ignição seja desligada. 

  • O atraso de vídeo configura o tempo que o equipamento continuará gravando as filmagens no SDcard após a ignição ser desligada. Este tempo sempre deve ser inferior ao tempo de retardamento.
  • O tempo definido na hora de luz, configura por quanto tempo o monitor conectado no equipamento continuará exibindo imagem após a ignição ser desligada.

 

¶ Definição de Proteção Contra Bateria Baixa

Preferências → Definição Básica → Definição ao Ligar/Desligar → Repouso

Nesta tela você pode ativar a configuração de proteção contra bateria baixa, ao ativar essa proteção, o equipamento entrara em um modo sleep quando a tensão de alimentação ficar a baixo do definido no primeiro parâmetro. Com o equipamento acordando do sleep após a tensão de alimentação subir acima do configurado no segundo parâmetro.

  • É possível também configurar para que um alarme seja disparado quando a tensão baixa for detectada.

 

¶ Configuração de Volume

Preferências → Definição Básica → Definição de Volume

Nesta tela é possível ajustar os volumes das vozes emitidas pelo equipamento.

 

¶ Configuração de Hodômetro

Preferências → Coleção de Dados → Definição Normal → Quilometragem

Nesta tela é possível verificar a distância atual do odômetro do equipamento, e definir um offset para o odômetro inicial, ao alterar o campo “Valor de base da quilometragem”.

 

¶ Definições de Usuário

Preferências → Definição Básica → Definição de Usuário

Nesta tela é possível configurar os logins de usuário.

  • É possível definir um horário de saída, onde o usuário é automaticamente deslogado do equipamento, após o tempo configurado.
  • A conta admin é capaz de adicionar até 2 usuários com permissões apenas de consulta.
  • Clicando no ícone é possível editar os dados do usuário.

 

¶ ? Configuração de Câmera

Preferências → Definição de Câmera

Nesta tela é possível ajustar o ângulo de instalação, virar e espelhar a imagem.

 

A fim de a imagem da câmera na orientação correta independentemente do local de instalação.

 

¶ ? Configurações de CHIP e APN

Preferências → Definição de Rede → Módulo de Comunicação

Nesta tela é possível configurar a APN do chip utilizado, insira o endereço da APN no campo “Ponto de acesso” e o usuário e senha nos seus respectivos campos.

  • É possível também definir se o equipamento irá utilizar os protocolos IPV4, IPV6 ou ambos.
  • Não é necessário alterar os dados nos demais campos

 

¶ ? Configuração Wifi

Preferências → Definição de Rede → WIFI

Nesta tela é possível configurar o módulo Wifi do equipamento para funcionar no modo cliente ou AP.

 

Modo cliente – Neste modo os campos ESSID e senha definem o nome e senha da rede Wifi que o equipamento se conectara para transmitir seus dados ao servidor. O campo “Duração do modo AP” define por quanto tempo o equipamento deixara o AP ativado após a ignição ser ligada, possibilitando a configuração do equipamento antes que ele se conecte na rede Wifi.

Modo AP – Neste modo os campos ESSID e senha configuram o nome e senha da rede que o equipamento irá abrir, para que outros dispositivos se conectem a sua rede.

 

¶ ? Configuração de Reporte GPS

Preferências → Coleção de Dados → Definição Normal → Localização

Nesta tela é possível alterar o intervalo e condições que o equipamento irá enviar um reporte de GPS a plataforma. É possível configurar o reporte de duas formas: 

Modo de reporte fixo – Neste modo o equipamento irá reportar sua posição em intervalos fixos de tempo e/ou distância, conforme configurado nos campos de tempo e distância.

 

Modo de reporte condicional – Neste modo é definido uma velocidade que indica o início da movimentação do veículo, e uma velocidade que indica o fim da movimentação do veículo. Quando o veículo está com a velocidade acima da velocidade configurada no campo “Ponto inicial de movimento” um contador é iniciado com o valor configurado no campo “Duração”, ao contador ser zerado, o equipamento passara a reportar sua posição de acordo com o intervalo definido nos campos de tempo e distância abaixo do campo de duração. Quando a velocidade do veículo fica abaixo da configurada no campo “Ponto final de movimento” um contador com o valor definido no segundo campo de duração é iniciado, após o contador ser zerado, o equipamento passara a reportar a sua posição de acordo com o intervalo definido no campo de tempo, abaixo do segundo campo de duração.

Desta forma é possível definir um intervalo de reporte quando o veículo é considerado estar em movimento, e um outro intervalo, quando o veículo é considerado parado, e definir as condições para que ele seja considerado em movimento ou parado.

 

Reporte por ponto de inflexão - Este é um tipo de reporte que pode ser utilizado em conjunto com um dos outros dois modos, nele é possível configurar um tempo, e um ângulo de inflexão, quando ocorre uma mudança no ângulo de direção do veículo igual a configurada, um reporte de posição é enviado a plataforma caso a mudança permaneça pelo tempo especificado. 

 

¶ ?️ Configuração de Servidor

Preferências → Definição de Rede → Definição de Servidor

Nesta tela é possível adicionar até 4 servidores que o equipamento se comunicara para transmitir seus dados.

  • O botão + adiciona um servidor a lista enquanto o botão X remove um servidor da lista, sendo que o último servidor não é possível ser removido.
  • O botão ativar define se o equipamento irá transmitir ou não dados com aquele servidor.
  • É possível habilitar que os dados trocados pelo equipamento e servidor sejam criptografados pela tecnologia TLS, porém é necessário que o servidor tenha integrado receber os dados dessa forma.
  • Na opção “Habilitar rede” é possível escolher se o equipamento transmitira os dados para este servidor através da comunicação 4g/3g por WiFi, ou ambos, escolhendo de maneira automática.

 

Aqui ficam os campos para configuração de endereço do servidor principal e do servidor de mídia, que podem ter o mesmo endereço.

No campo da porta é configurado a porta TCP e a porta TLS caso a criptografia esteja habilitada.
